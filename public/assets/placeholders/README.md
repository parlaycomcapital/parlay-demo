# Placeholder Images

This directory contains placeholder images for development and demo purposes.

## Required Images

- `placeholder.png` - Generic placeholder image (400x200 recommended)
- `avatar.png` - User avatar placeholder (48x48 or square)
- `post-image.png` - Post content image placeholder (600x400 recommended)
- `group-avatar.png` - Group/community avatar placeholder (64x64 or square)

## Usage

These images are used when `PLACEHOLDER_MODE=true` or when actual images are not available.

You can replace these with your own placeholder images or use online placeholder services:
- https://via.placeholder.com/400x200
- https://placehold.co/400x200
